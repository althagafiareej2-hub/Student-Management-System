# Student-Management-System

## Description
This project is a simple web application that manages student information using HTML, CSS, JavaScript, PHP, and MySQL.

The website allows users to enter student names and ages, save the data in a database, display the records in a table, and change the student status using the Toggle button without refreshing the page.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- InfinityFree Hosting

## Project Files

The project contains the following files:

- `index.php`  
  The main page that includes the student form and displays database records.

- `insert.php`  
  Saves the entered student information into the MySQL database.

- `toggle.php`  
  Updates the student status between 0 and 1 using the Toggle button.

- `script.js`  
  Handles dynamic updates using JavaScript Fetch API without reloading the page.

- `style.css`  
  Contains the website design and styling. The interface was customized using purple and violet gradients, with a styled form, buttons, and table.

- `README.md`  
  Contains project documentation and instructions.

## Deployment and GitHub Upload

The project files were uploaded and deployed on **InfinityFree** hosting to make the website accessible online.

The database connection file was handled separately because it contains sensitive information. The original `db.php` file was replaced before uploading the project to GitHub to protect database credentials and prevent exposing private information.

The website was successfully connected with a MySQL database using InfinityFree hosting.

## Final Result

The project provides:
- Adding student names and ages.
- Saving data in a MySQL database.
- Displaying stored data in a table.
- Changing student status using the Toggle button without refreshing the page.
- A customized interface with a purple-themed design.
